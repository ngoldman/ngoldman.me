Projects

  • Code for Portland

    Code for Portland is a Code for America brigade that I started in March of 2014 with lots of help from Erica Lauer Vose, Nathan Wittstock, and Mele Sax-Barnett. We host civic hacking events, work on civic tech projects with municipal partners and non-profits, and advocate for open data.

    Site Meetup Group Github Org Twitter

  • Geobin.io

    Geobin was a project I started at work based on an idea from Aaron Parecki. It allows you to inspect HTTP requests with geographic data in real time. Originally an internal debugging tool for the Esri Geotrigger Service, it grew into an open source project to test the limits of AngularJS, Golang, and WebSockets with some major help from Ryan Arana, Court Fowler, and Josh Yaganeh.

    Site Docs Source

  • TinyStore

    TinyStore is a tiny web storage wrapper (< 1k minified). It works with localStorage, sessionStorage, objects, and functions (hat tip to Jerry Sievert for expanding support beyond localStorage). I originally wrote it for use with Geobin. My goal was to create the simplest possible API for persisting session information.
