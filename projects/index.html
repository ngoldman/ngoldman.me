Projects

Koop

Koop is an open source project started by Chris Helm and the Esri DC R&D Center to create a modular ETL engine for dynamically transforming geospatial data on the web. I took over management of the project after Chris moved on from Esri, then handed my role off to Daniel Fenton after I left EsriPDX.

Friends

Friends is like Slack except P2P, offline friendly and open source. We transmit data over secure channels and authenticate messages. It’s made by a group of open source developers who are motivated to make a public chat platform that is easy to use but isn’t centrally controlled, and where data is owned by its users.

Code for Portland

Code for Portland is a Code for America brigade I founded with help from Erica Lauer Vose, Nathan Wittstock, and Mele Sax-Barnett in 2014. We hosted civic hacking events, worked on civic tech projects with municipal partners and non-profits, and advocated for open data. I’m currently looking to hand off management of the group to someone excited about being a local leader in civic technology.

ArcGIS for Developers

I worked on the design and development of a documentation site for developers that needed to use Esri’s ArcGIS web platform, along with Paul C Pederson, Nikolas Wise, Rachel Nehmer, Manny Lopez, and our fearless leader Patrick Arlt.

geobin.io

A web application that allows you to inspect HTTP requests with geographic data. Built at the Esri Portland R&D Center with Ryan Arana, Josh Yaganeh, and Court Fowler.

Geotrigger Service

A web platform and set of SDKs for adding geofencing capabilities to mobile applications. I helped build supporting software for it like the Geotrigger Editor and Geotrigger Faker.
