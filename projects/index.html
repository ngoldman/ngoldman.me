Nate Goldman projects

Projects

geobin.io

A web application that allows you to inspect HTTP requests with geographic data. Built at the Esri Portland R&D Center with Ryan Arana, Josh Yaganeh, and Court Fowler.

Koop

Working with Chris Helm and the Esri DC R&D Center to create a modular ETL engine for viewing and downloading open spatial data from different providers and formats. Pull requests welcome.

Geotrigger Service

A web platform and set of SDKs for adding geofencing capabilities to mobile applications. I helped build supporting software for it like the Geotrigger Editor and Geotrigger Faker.

ArcGIS for Developers

I work on the design and development of a documentation site for developers that need to use Esri’s ArcGIS web platform, along with Paul C Pederson, Nikolas Wise, Rachel Nehmer, Manny Lopez, and our fearless leader Patrick Arlt.

Many more small projects at github.com/ngoldman

Orgs

Code for Portland

A Code for America brigade I started with help from Erica Lauer Vose, Nathan Wittstock, and Mele Sax-Barnett in 2014. We host civic hacking events, work on civic tech projects with municipal partners and non-profits, and advocate for open data. Currently we’re focusing on housing.

Esri PDX

A small team of geniuses brought together by Amber Case and Aaron Parecki working on web and mobile geospatial tools. I work there as a web developer and designer. My secret agenda is to bring Open Web values and processes to the monolithic mapping software corporation that we serve.
